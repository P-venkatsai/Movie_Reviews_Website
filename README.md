# Movie Reviews Website

Welcome to the Movie Reviews Website! This website allows users to give reviews of recently released movies. You can access the website [here](https://p-venkatsai.github.io/Movie_Reviews_Website/).

## Features

### 1. Main Page (index.html)
The main page is the landing page of the website and contains information about the recently released movies. It provides an overview of various movies with their details such as title, release date, and rating. The main page also includes a navigation bar with the following buttons:

- **Sign In**: Clicking this button will allow users to sign in to their accounts.
- **Sign Up**: Clicking this button will direct new users to the registration page to create an account.
- **Search**: By clicking this button, users can search for a specific movie by entering its title or other relevant details.
- **Movie Box**: Clicking on any movie box will redirect users to the movie details page.

### 2. Movie Details Page (movie-details.html)
The movie details page provides comprehensive information about a specific movie. It includes details such as title, release date, duration, genre, director, and a brief synopsis. Users can also give reviews for the movie on this page. This feature allows users to share their opinions and ratings, helping others make informed decisions.

### 3. Movie Collection (movie-collection.js)
The movie collection page presents a collection of movies of a certain type or genre. It provides an organized view of movies based on specific criteria, making it easier for users to explore and find movies of interest.

### 4. Sign In and Sign Up
The website includes a sign-in and sign-up functionality, allowing users to create accounts and log in to access additional features. By signing in, users can save their preferences, track their reviewed movies, and interact with other users.

### 5. Search Functionality
The search button on the navigation bar enables users to search for movies based on various criteria, such as title, genre, or actor. This feature helps users quickly find specific movies or discover new ones.

### 6. User Reviews and Ratings
Users have the ability to provide reviews and ratings for movies they have watched. This feature promotes engagement and allows users to share their opinions about movies, helping others make informed decisions.

## Usage

To use this website, follow these steps:

1. Access the website using the provided link: [Movie Reviews Website](https://p-venkatsai.github.io/Movie_Reviews_Website/).
2. On the main page, browse through the list of recently released movies.
3. If you want more information about a specific movie or wish to provide a review, click on the movie box to navigate to the movie details page.
4. On the movie details page, you can find detailed information about the movie, including its title, release date, duration, genre, director, and a brief synopsis. You can also give your review and rating for the movie.
5. You can also use the search button on the navigation bar to search for a specific movie by entering its title, genre, or actor's name. This will help you quickly find movies of interest.
6. If you want to explore movies of a specific type or genre, navigate to the movie collection page. This page provides an organized view of movies based on specific criteria, making it easier for you to find movies that match your preferences.
7. If you wish to save your preferences, track your reviewed movies, or interact with other users, you can create an account and sign in to access additional features.
